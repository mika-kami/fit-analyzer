# Mock FIT Files

Generated activity FIT files for planning tests:

- `mock_base.fit` — long low-intensity endurance session (mostly Z2)
- `mock_aerobic.fit` — sustained aerobic/tempo load
- `mock_anaerobic.fit` — interval-heavy high-intensity session

## Regenerate

```bash
npm run mock:fits
```

Generator script:

- `scripts/generate-mock-fits.mjs`
